    منصة AhmedETAP للهندسة الكهربائية — نظرة عامة

    مقدمة

    منصة AhmedETAP للهندسة الكهربائية هي نظام هندسي ذكي متكامل وجاهز للإنتاج، مصمم لإجراء دراسات أنظمة الطاقة الكهربائية بشكل مستقل. تجمع المنصة بين محركات الحساب البرمجية الأصلية بلغة بايثون، ونظام تنسيق الوكلاء الذكاء الاصطناعي، وأتمتة برنامج ETAP عبر واجهة COM، وإثراء البيانات الجغرافية المكانية، والأمان على مستوى المؤسسات.

    تم تصميم المنصة لتكون أداة شاملة للمهندسين الكهربائيين، حيث يمكنهم إجراء جميع أنواع الدراسات الهندسية من تدفق القدرة إلى تحليل الوميض القوسي في بيئة واحدة متكاملة، مع ضمان الامتثال للمعايير الدولية مثل IEEE و IEC و NFPA.

    الميزات الرئيسية

    ⚙️ الدراسات الهندسية

    توفر المنصة مجموعة شاملة من الدراسات الهندسية لأنظمة الطاقة الكهربائية، كل منها مُطابق للمعايير الدولية المعتمدة:

    الدراسة المحرك الحسابي المعيار
    تدفق القدرة نيوتن-رافسون، الفصل السريع، التدفق الأمثل المستمر IEEE 141, IEEE 399
    دائرة القصر تحليل الأعطال حسب IEC 60909 IEC 60909, IEEE C37.010
    الوميض القوسي حساب الطاقة الحادثة ومستوى معدات الحماية الشخصية IEEE 1584-2018, NFPA 70E
    التوافقيات تحليل التوافق التوافقي الكلي والجزئي IEEE 519-2022
    التدفق الأمثل التدفق الأمثل للقدرة المتناوب والمستمر مع التوزيع الاقتصادي IEEE 3002.2
    تنسيق الحماية تنسيق منحنيات الوقاية حسب IEC 60255 IEC 60255, IEEE C37.112

    🧠 نظام الوكلاء الذكي

    تعتمد المنصة على نظام متعدد الوكلاء يضم ١٥ وكيلاً متخصصاً ينسق بينهم وكيل رئيسي (Chief Engineering Orchestrator):

    1. وكيل تدفق القدرة — تحليل تدفق القدرة بطرق نيوتن-رافسون والفصل السريع
    2. وكيل دائرة القصر — حساب تيارات القصر حسب IEC 60909
    3. وكيل التوافقيات — تحليل التوافقيات والتحقق من IEEE 519-2022
    4. وكيل التدفق الأمثل — التدفق الأمثل للقدرة مع التوزيع الاقتصادي
    5. وكيل تنسيق الحماية — تنسيق الوقايات حسب IEC 60255
    6. وكيل تنفيذ ETAP — أتمتة ETAP عبر واجهة COM
    7. وكيل التحقق — التحقق المتقاطع من النتائج
    8. وكيل التقارير — إنشاء تقارير PDF و DOCX و XLSX
    9. وكيل الاستقرار — تحليل الاستقرار العابر والصغير حسب IEEE 399
    10. وكيل أحجام الكابلات — حساب أحجام الكابلات حسب IEC 60364
    11. وكيل شبكات التأريض — تصميم شبكات التأريض حسب IEEE 80
    12. وكيل الطاقة المتجددة — تحليل تكامل الطاقة المتجددة حسب IEEE 1547
    13. وكيل تخزين البطاريات — تحليل أنظمة تخزين الطاقة حسب IEC 62933
    14. وكيل SCADA — التكامل مع أنظمة SCADA حسب IEC 61850

    🔌 أتمتة ETAP

    تدعم المنصة التكامل المباشر مع برنامج ETAP عبر واجهة COM للأتمتة، مما يسمح بتشغيل الدراسات تلقائياً والتحقق المتقاطع من النتائج بين المحرك الأصلي وبرنامج ETAP.

    🗺️ التكامل الجغرافي المكاني

    توفر المنصة تكاملاً مع أنظمة المعلومات الجغرافية (GIS) بما في ذلك ArcGIS و QGIS، مع التحقق من صحة البيانات الطوبولوجية والسمات الكهربائية.

    🛡️ الأمان على مستوى المؤسسات

    تتضمن المنصة طبقات أمان متعددة:

    • المصادقة: JWT مع bcrypt (تكلفة ١٤) وقفل الحساب (٥ محاولات) وتشفير Fernet
    • التفويض: التحكم في الوصول القائم على الأدوار مع ٥ أدوار (مسؤول، مهندس، محلل، مشاهد، ضيف) و ٢٥+ صلاحية
    • الحماية: بيئة Python آمنة مع التحقق من AST والمهلات الزمنية
    • إدارة الأسرار: HashiCorp Vault مع نسخ احتياطي محلي مشفر
    • التدقيق: سجل تدقيق منظم بصيغة JSON
    • تحديد المعدل: خوارزمية دلو الرموز مع تتبع لكل عميل

    البنية التقنية

    الطبقة التقنية
    الخلفية Python 3.13+، FastAPI، Pydantic v2
    الواجهة React 19+، TypeScript 5.7+، Mastra، Tailwind CSS 4
    الذكاء الاصطناعي LangChain، RAG، ١٥ وكيل متخصص
    قاعدة البيانات PostgreSQL، Redis (التخزين المؤقت)، Redis Queue
    العمليات Docker، Docker Compose، GitHub Actions
    الأمان JWT، bcrypt، HashiCorp Vault، Trivy، CodeQL
    المعلومات الجغرافية ArcGIS API، QGIS، GDAL
    التكامل ETAP COM (Windows)، BIM UDM، SCADA
